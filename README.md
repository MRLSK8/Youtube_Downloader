# Youtube_Downloader

Required:
  * Python 3
  * pip
  * pytube

# Getting started
  ### Clone the repository
   * git clone https://github.com/MRLSK8/Youtube_Downloader.git
  ### Inside the folder install *pytube*
   
   * pip install pytube

Now you're able to run the program. :ok_hand:
